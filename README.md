Application de gestion des salles


Nom :Taib
Prénom :Rahma

Description du projet

Ce projet consiste à développer une application en Python pour la gestion des salles.

L’application permet de :

Ajouter une salle
Modifier une salle 
Supprimer une salle
Rechercher une salle
Afficher la liste des salles

Technologies utilisées

Python
MySQL
Git & GitHub

Architecture

Le projet est organisé en couches :

Data (DAO)
Models
Services
Views (GUI)

Fonctionnalités

Connexion à une base de données MySQL
Gestion CRUD complète des salles
Interface graphique simple
